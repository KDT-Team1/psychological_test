# psychological_test
심리테스트 제작해보기
<br><br>

> ### 제작 목표
- DB 없이 구현할 수 있는 반응형 웹을 제작하려 함
- 요즘 인기 있는 주제인 심리 테스트 페이지를 선택
<br>

> ### 설치 방법
```
git clone https://github.com/KDT-Team1/psychological_test.git
npm install express
npm install ejs
node app.js
```
- localhost:8080 접속
